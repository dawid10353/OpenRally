# OpenRally

**OpenRally** is an open-source 3D rally game built directly in the browser. 

The primary goal of this project is to build an amazing, fully-featured open-source game developed **exclusively with the support of Artificial Intelligence (AI)**. From generating 3D models to writing the core game engine and physics, AI agents are the driving force behind OpenRally's development.

## 🏎️ About the Game

OpenRally brings the thrill of off-road racing to your web browser:
- **Dynamic Terrain:** Drive your vehicle across uneven, open terrain featuring procedurally generated hills, valleys, and elevations.
- **Arcade-Sim Physics:** Experience a satisfying blend of arcade fun and simulation mechanics. Master drifting and feel the suspension work as you navigate rough tracks.
- **Immersive Environment:** Enjoy dynamic lighting, post-processing effects, particle effects like dust, and realistic engine sounds.

## 🚀 How to Run

Running the game locally is quick and easy.

1. **Install dependencies:**
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Play:**
   Open the address shown in your console (usually `http://localhost:5173`) in your web browser.

## 🗺️ Roadmap

### Stage 1 — Foundation (COMPLETED)
- Terrain generation, core physics, camera systems, basic controls, HUD, and lighting.

### Stage 2 — Polishing (IN PROGRESS)
- Particle effects (dust), engine sound, different surface types, map objects, and advanced post-processing.

### Stage 3 — Expansion
- Integrating AI-generated 3D models (GLB format), replacing placeholder shapes with high-quality vehicles and environment assets, optimization, and refactoring.

### Stage 4+ — Future
- Multiplayer modes, a custom map editor, dynamic weather systems, and full automation of asset generation.
